FotoRama

A simple yet powerful JQuery plugin

WIA Class Presentation By: Miranda Johnson

Features

  1. Supports Auto play
  2. Fullscreen
  3. Responsive
  4. Keyboard navigation
  5. Transitions
  6. Callback events
  7. Supports video, image, html & text
  8. Cross Browser compatibility

Three simple steps

You can have your gallery up and running

In Three simeple steps !

Set Up Instructions

  • Link to jquery (1.8 or later)
  • Link to fotorama.css & fotorama.js
  • Add Class=fotorama to your Div

FotoRama Code


							

							<!-- 1. Link to jQuery (1.8 or later), -->

							<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script> <!-- 33 KB -->

							

							<!-- fotorama.css & fotorama.js. -->

							<link  href="http://cdnjs.cloudflare.com/ajax/libs/fotorama/4.6.2/fotorama.css" rel="stylesheet"> <!-- 3 KB -->

							<script src="http://cdnjs.cloudflare.com/ajax/libs/fotorama/4.6.2/fotorama.js"></script> <!-- 16 KB -->

							

							<!-- 2. Add images to <div class="fotorama"></div>. -->

							<div class="fotorama">

							<img src="http://s.fotorama.io/1.jpg">

							<img src="http://s.fotorama.io/2.jpg">

							</div>

							

							<!-- 3. Enjoy! -->-->
